Voici une première version d'un programme développé très rapidement
par un débutant en Python. Le programme effectue une recherche dans un
dossier "dir" et vérifie dans chaque fichier si la chaine de caractère
"hello" est présente. On vous demande de tester le programme et de
corriger le code source afin que celui ci soit digne d'un
ingénieur. Le programme sera vérifié par des experts en
Python. Celui-ci doit donc être d'une qualité supérieur. Le client
aimerai également être en mesure d'utiliser le programme sous forme
API (programmatique), et pas seulement comme un programme standalone (ligne de commande),

Bien que le programme soit fonctionnel, il contient plus d'une
quinzaine d'erreurs ou de choses à corriger. Cela peut être n'importe
quoi:

* Mauvaise utilisation du language
* Oubli de bonnes pratiques Python
* Mauvaise gestion des erreurs
* Mauvaise implémentation
* Ne pas utilisez les modules standard Python
* Problèmes potentiels de performance (utilisation mémoire etc.)

Vous pouvez faire tous les changements que vous voulez au programme.

Bonus: Compatible Python 3.
